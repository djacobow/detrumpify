# Detrumpify

A browser extension for Chrome and Android Firefox that replaces instances of names of Trump administration figures and allies with more apt descriptors.

The initial batch of Trump insults came from:
http://theslot.jezebel.com/in-honor-of-donald-trumps-70th-birthday-here-are-70-wa-1781899319

## Features

* Insults loaded from an external config file (refreshed weekly)
* Clean and NSFW insult modes, selectable via the extension button
* Insults can be enabled/disabled per person
* Trump images replaced with something more appropriate
* User can supply a custom config URL

## Installation

### Chrome Web Store
https://chrome.google.com/webstore/detail/detrumpify/flekeckcgklibaplikocggegkhgaimea

### Developer Mode (Chrome)
Clone the repo and load the extension at `chrome://extensions` with developer mode enabled.

### Android Firefox
Load as a temporary extension via `about:debugging`, or install from the AMO listing.

## Config Generation

Insult configs are generated by `configs/make_variations_v2.py` and served from:
`https://toolsofourtools.org/detrumpify2/v2/`

Available configs:
* `combined.json` — clean + NSFW insults
* `clean.json` — clean insults only
* `dirty.json` — NSFW insults only
* `disable.json` — temporarily disable all replacements
